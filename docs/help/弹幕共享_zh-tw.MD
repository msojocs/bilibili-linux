# 彈幕共享

彈幕共享是什麼？

彈幕共享是一個提供不同區域彈幕互通的服務。

## 場景

你在港澳台或東南亞觀看某一部番劇，而它的彈幕少得可憐，同時你在沒有彈幕的情況下對此番提不起興趣；

於是，你想："可不可以把其它地方的彈幕調過來用？"

> 為什麼要調過來？
> 1. 其它地方可能畫質欠佳
> 2. 其它地方可能加載視頻緩慢
> 3. 其它地方可能沒有視頻資源
